#### **zerodown-server**

## Quick Start Guide

Setting up Virtual Environment

    py -3 -m venv venv

Activating Virtual Environment

    venv\Scripts\activate

Installing Required Packages

    pip install -r requirements.txt

Starting the Server

    python wsgi.py
