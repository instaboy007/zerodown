## Quick Start Guide

Git Clone this Repository using

    git clone https://github.com/instaboy007/zerodown.git 

 Install Node modules using
 
    npm install
    
Go to the Directory zerodown by using the command
    
    cd zerodown
    
Run the App using

    npm start
